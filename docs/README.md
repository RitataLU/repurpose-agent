## What we found
An autonomous scoring agent analyzed 169 GWAS-implicated genes with no approved drug
across 6 diseases. Using a Spearman correlation metric against 13 curated gold-standard
gene-drug pairs, the agent iterated 10 scoring formulas to find which biological features
best predict drug target potential.
