The Problem

Most GWAS hits have no approved drug

Genome-wide studies identify hundreds of disease-associated genes. Only a small fraction become drug targets. The rest are orphans — biologically compelling, commercially untested.

How It Works

Three phases, fully automated

🎯

Phase 1 — Gold Standard

Open Targets GraphQL API identifies genes with approved drugs AND GWAS genetics evidence per disease. These become the evaluation ground truth — discovered, never hardcoded.

🔍

Phase 2 — Orphan Genes

GWAS Catalog REST API yields genome-wide significant associations (p < 5×10⁻⁸) per disease. Genes with GWAS signal but no approved drug are the repurposing candidates.

⚗️

Phase 3 — Feature Enrichment

Each orphan gene is enriched with Open Targets druggability, PubMed activity, and proxy features for eQTL effect, Mendelian randomization z-score, tissue specificity, and PPI degree.

Autoresearch Loop

Agent iterates the scoring formula

agent.py runs 10 pre-defined experiments, keeping changes that improve recall@K and discarding the rest — then pushes results to GitHub.
